# Pank v2.0.0 Specification

## Version Name

Pank v2.0.0 "Foundation"

## Goal

Turn Pank from a single-purpose purge logger into a modular Discord bot framework for Stuniverse.

## Branches

- main = live production bot
- develop = active development
- pank-v1-backup = backup of the first working version

## Core Modules

- Core
- Logging
- Moderation
- Tickets
- Welcome
- Quiz
- Community
- AutoMod

## First Release Scope

v2.0.0 will include:

- Modular folder structure
- Event loader
- Config validation
- Logger
- Message cache
- Bulk purge logging
- Automatic deployment plan
- Documentation

## Deployment Rule

Only main should deploy to the live VPS.
